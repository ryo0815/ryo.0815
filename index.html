デモ版について

この GitHub Pages版は静的デモです。実際のAPI機能（画像認識、データベース操作）は動作しません。

完全な機能を体験するには、Node.jsサーバー環境が必要です。

Book Lending Vision

画像認識技術を活用した次世代の図書貸出管理システム。 書籍の表紙を撮影するだけで、瞬時に貸出・返却処理が可能です。

Google Cloud Vision API Airtable Database Responsive Design

システム機能

簡単3ステップで図書館業務を効率化

書籍の貸出

書籍の表紙を撮影し、Google Cloud Vision APIで書名を自動認識。 生徒名を入力するだけで簡単に貸出処理が完了します。

書籍の返却

返却する書籍を撮影し、返却期限をチェック。 遅延の有無を確認して適切な返却処理を実行します。

延長申請

返却期限の2日前から延長申請が可能。 現在の貸出状況を確認し、7日間の延長処理を実行します。
